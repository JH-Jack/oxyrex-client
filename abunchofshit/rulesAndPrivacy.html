Oxyrex.io - Rules and Privacy
Rules | Privacy Information | Contact Us

These rules are required by all players to be followed at all times while using any service provided by Oxyrex.io. Failure to comply with these rules can result in anything from a kick, to a temporary ban to a permanent ban.

  1. Do not attempt to modify the game in any way to change any aspects of the game that can automate or change any skill based or unintentional features using scripts or other procedures. Changing visual process that do not affect gameplay such as color and UI positioning is allowed.
  2. A bug is an unintentional aspect of the game that is not an intended feature. Do not abuse or exploit bugs. If you find a bug, please report it on the feedback page.
  3. Connecting to the game servers using any method that is not joining via the Oxyrex.io client is not allowed. This includes botting of any form.
  4. Harassing anyone through any means, such as naming yourself something to offend another user, sending rude or offensive messages using the broadcast commands of the In-game terminal or the Discord bot is not allowed.
  5. No intentionally lagging or crashing servers by any means.
  6. The Developers are the law of the game, if we tell you to stop doing something, then you are required to do so or face consequences.

We know that data collection is important to you, so it is our goal to be as transparent as possible. We do collect your IP address when you join our game servers. Once your IP is read from the WebSocket information, it is immediately hashed using SHA256, and only used to stop multiple tabs being opened, and to block banned users. If you end up getting banned from the game, your hashed IP address will be saved to a secure database. We never deal with raw IP addresses. When you connect to the main site, your IP is again hashed and only used for differentiating between users for seeing how many different people join the game within time periods. In either case, your IP is never seen by anyone and is encrypted with SHA256, which is a secure hashing algorithm to protect your data. Unless you are banned, your IP is never saved. Even when you are banned, the hash of the IP address is stored. We block all VPN or Proxy IP connections, and you will be forced to disable the VPN or Proxy in order to join.
